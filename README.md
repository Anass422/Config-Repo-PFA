# AutoDrive AI Platform - Configuration Repository

Repository de configuration centralisée pour tous les microservices de la plateforme AutoDrive.

## Services Configurés

- gateway-service (8080)
- auth-service (8081)
- vehicle-service (8082)
- order-service (8083)
- payment-service (8084)
- detection-service (8085)
- notification-service (8086)

## Utilisation

Ce repository est utilisé par le Config Server pour fournir les configurations à tous les microservices.
